## 🌌 The Vision
This isn't just a static resume; it’s a full-stack exploration of modern web sensory design. I built this to bridge the gap between **Intelligent Systems** and **Immersive UI**. Every interaction—from the 3D hardware-accelerated tilt cards to the live Firebase guestbook—is engineered to provide a seamless, "living" user experience.

---

## 🚀 Key Features

### 🧠 Intelligent Integrations
- **Live GitHub Analytics:** Real-time fetching of repository counts and follower stats using the GitHub REST API.
- **Firebase Cloud Database:** A persistent, real-time Guestbook where visitors can leave a mark that stays pinned for everyone.
- **"Vaibot" Mini-AI:** A custom-built chatbot logic to handle FAQ and provide quick access to my credentials.

### 🎮 Sensory & Physics Engine
- **CSS3 Physics:** Repelling skill tags that calculate mouse proximity to simulate magnetic force.
- **3D Tilt Architecture:** Cards that respond to mouse coordinates to create a hardware-accelerated depth effect.
- **Cinematic Sequencing:** An anime-inspired (Record of Ragnarok/Luffy/Gojo) splash screen transition that "explodes" into the main UI.
- **Ambient Lofi Environment:** Integrated background audio with a global mute state saved to `localStorage`.
