## ⚡ Core Telemetry & Features

### 🛰️ Serverless Intelligence
* **Orbital Telemetry HUD:** Real-time fetching of GitHub repository counts and follower stats directly into a raw terminal readout.
* **Firebase Cloud Database:** A persistent, real-time Guestbook where visitors can leave a mark that stays pinned to the cloud.
* **"Vaibot" AI Interface:** Custom-built chatbot logic to handle FAQs, parse intent, and provide quick access to my credentials.

### 🕳️ Sensory & Physics Engine
* **Hyperdrive Scroll Engine:** Floating starfields that stretch into light-trails based on your real-time scroll velocity, perfectly synced with a breathing, mouse-tracked parallax Nebula.
* **Holographic UI & Circuit Tracing:** Hardware-accelerated 3D tilt cards featuring SVG neon circuit-tracing on hover and raw data-packet text scrolling.
* **Audio-Reactive Environment:** Integrated Lofi background audio tied directly to the UI—when the beat drops, the holographic profile and ambient orbs pulse in sync.
* **CSS3 Magnetic Force:** Repelling skill tags that dynamically calculate mouse proximity to simulate magnetic repulsion.
